
# Online Sales Analysis

## Project Description

Python OOP project for managing products and customer carts.

## Classes

### Product
Represents a product with:
- name
- price
- quantity

### ProductManager
Handles:
- adding products
- displaying products
- calculating inventory value
- removing products

### Cart
Handles:
- adding products to cart
- displaying cart items
- calculating total payment
